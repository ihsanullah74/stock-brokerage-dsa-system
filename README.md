# 📈 Stock Brokerage & Management System (DSA Project)

A web-based application built using **Python Flask** and **HTML/CSS/JavaScript** to demonstrate real-world applications of **Data Structures and Algorithms (DSA)**.

---

## 🚀 Features

- Add stock entries (company name, symbol, price)
- Store data using **Binary Search Tree (BST)**
- Sort stocks using **Merge Sort**
- Search stocks using **Binary Search**
- Undo recent updates using **Stack**

---

## 🧠 DSA Concepts Used

| Concept | Purpose |
|--------|--------|
| Binary Search Tree | Efficient storage and retrieval |
| Merge Sort | Sorting stock data |
| Binary Search | Fast searching |
| Stack | Undo functionality |

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **Algorithms:** BST, Merge Sort, Binary Search, Stack

## ⚙️ How It Works

1. User enters stock data
2. Data is stored in a Binary Search Tree
3. Merge Sort organizes data for display
4. Binary Search finds specific stocks quickly
5. Stack stores recent updates for undo operations
